To install the packages, follow these steps.

WINDOWS MACHINE
---------------
	-> unzip the folder
	-> run cmd to the select path that includes setup.py
	-> code
		python setup.py install


IOS MACHINE
-----------
	-> unzip the folder
	-> run python bash
	-> code python setup.py install
	-> write down the path where package has been installed (i.e. "C:\Python27\Lib\site-packages\")
	-> include on python editor (i.e. Spyder) the above path using its managing tool
	-> Update Module Names List
	-> Close and restart the editor
